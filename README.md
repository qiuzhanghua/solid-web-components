## Web component with SolidJS

## install

```bash
pnpm install
```

## dev
```bash
pnpm dev
```

## build
```bash
pnpm build
```
## add .d.ts (optional)
```bash
pnpm dts
```

## publish
```bash
pnpm publish
```
